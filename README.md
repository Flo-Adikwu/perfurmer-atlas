# The Perfumer's Atlas

An interactive fragrance note pairing guide designed for fragrance enthusiasts and aspiring perfumers.

## Features

- **Note Browser**: Explore 40+ fragrance notes across Top, Heart, and Base layers.
- **Search & Filter**: Find notes by name, family, or layer.
- **Note Details**: Learn about each note's olfactive family, character description, and fun facts.
- **Pairing Guide**: Instantly see which notes pair beautifully and which ones to handle with care.
- **Comparison Mode**: Compare up to 3 notes side-by-side to understand their combined profiles.
- **Favourites**: Save your favourite notes for quick access (stored locally).
- **Responsive Design**: Works beautifully on mobile and desktop.

## How to Run Locally

1. Ensure you have [Node.js](https://nodejs.org/) installed.
2. Open your terminal or command prompt.
3. Navigate to the project folder: `cd perfumers-atlas`
4. Install the dependencies: `npm install`
5. Start the development server: `npm run dev`
6. Open the URL shown in the terminal (usually `http://localhost:5173`) in your web browser.

## Aesthetic

Designed with a clean, botanical, and earthy aesthetic, inspired by natural perfumery handbooks.

- **Typography**: Playfair Display (Headings) & Inter (Body)
- **Color Palette**: Moss Green, Terracotta, and Warm Off-White

Enjoy exploring the art of fragrance!
